# opencode-template-gf

Template limpio de configuracion para OpenCode con memoria persistente via Obsidian.

## Incluye
- 21 skills preinstaladas
- 15 subagentes configurados
- 29 plugins
- MCP de Obsidian para memoria persistente
- MCP de Playwright y Stitch (sin API keys)

## Uso rapido
1. Copia \opencode.template.jsonc\ como \opencode.jsonc\
2. Edita los placeholders con tu ruta de vault de Obsidian
3. Ejecuta \
px opencode-marketplace install ...\ (ver IMPLEMENTATION.md)
4. Copia las skills a \~/.agents/skills/\
5. Ejecuta \
pm install\ en \mcp-servers/\

Ver \IMPLEMENTATION.md\ para guia detallada.
